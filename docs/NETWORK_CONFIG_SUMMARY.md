## ⚠️ 注意事项

### 相机功能限制
- 手机浏览器中的相机功能需要 HTTPS 或 localhost
- 如果相机无法启动，可能需要：
  - 使用 HTTPS 访问
  - 在电脑上使用 localhost 测试

### 防火墙设置
- 如果手机无法访问，请检查 Windows 防火墙
- 确保端口 3000 和 3001 未被阻止

### 网络要求
- 手机和电脑必须在同一个 WiFi 网络
- 确保网络允许设备间通信
